# Supply Chain Correlation Analysis

This repository contains the correlation analysis for key supply chain performance metrics of an automotive manufacturer, conducted as part of the Supply Chain Analytics assignment.

## Dataset Overview
The dataset includes 79 procurement transaction records with the following variables:

- **Supplier_Lead_Time** – Days from order placement to delivery  
- **Inventory_Levels** – Current stock levels (units)  
- **Order_Frequency** – Number of orders placed per month  
- **Delivery_Performance** – On-time delivery rate (%)  
- **Cost_Per_Unit** – Cost per unit in USD  

## Analysis Steps Performed
1. Imported the dataset into Excel  
2. Enabled the *Analysis ToolPak*  
3. Generated a **correlation matrix** using:  
   *Data → Data Analysis → Correlation*  
4. Applied **conditional formatting** (Red → White → Green) to create a correlation heatmap  
5. Exported the matrix to CSV  
6. Captured and saved the heatmap as an image
